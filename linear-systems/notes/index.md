    מערכת חסרת זיכרון

    $$h_{(t)} = k \cdot \delta _{(t)}$$
    $$h_{[t]} = k \cdot \delta _{[t]}$$

    סיבתיות
    
    $$h_{(t)} = 0 \ \ \ t<0$$
    $$h_{[n]} = 0 \ \ \ n<0$$

    התגובה להלם זה המוצא עבור 
    
    $$x_{[n]} = \delta _{[n]}$$
    ההלם מופיע ב
    
    $$n=0$$

    $$n=0$$
